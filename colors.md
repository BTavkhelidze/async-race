1. Main Background & Global Colors
   App Background: Deep Navy / Near-Black (#0A0E17) — Provides a high-contrast backdrop for the glowing accent elements.

Card / Track Backgrounds: Dark Charcoal (#151C2C) — Gives structure to the layout forms and the racing track blocks.

Borders / Dividers: Cool Slate (#1F293A) — Subtle, clean lines that keep the UI structured without feeling harsh.

2. Sidebar Navigation (Aside) Colors
   Sidebar Background: Deep Matte Black (#0A0E17 to matching gradient shadow).

Active Link Text ("Garage"): Vibrant Racing Orange (#FF5722) — Gives a bright focal point indicating where the user currently is.

Active Link Background Tint: Translucent Orange (rgba(255, 87, 34, 0.1)) — A soft background glow behind the text.

Inactive Link Text ("Winners", etc.): Muted Slate Blue (#64748B) — Recedes cleanly into the dark background, but remains perfectly legible.

Hover State Text: Soft Off-White (#CBD5E1) — Shifts bright enough to feel responsive when interacting.

3. Typography (Text Colors)
   Primary Headings: Crisp Platinum White (#E2E8F0) — Used for major text like "Garage (7 Cars, Page 1)".

Secondary Text / Subtitles: Dim Gray (#94A3B8) — Used for descriptions like "Manage your cars and start the race".

Disabled / Placeholders: Dark Slate (#475569) — Used for input placeholder texts and disabled buttons like "STOP".

4. Accent & Action Colors
   Primary Actions (Race, Create, START): Solid Racing Orange (#FF5722) — Demands immediate attention.

Secondary Actions (Reset, Select): Deep Navy with Orange Outline (border: #FF5722 at lower opacity or clear white text on grey background).

Destructive Actions (Remove): Steel Dark Gray (#1E293B) with muted text (transitions to a soft red highlight on hover).
